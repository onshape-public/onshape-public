Sample Code

This page provides some sample codes in a few different popular programming languages for you to get started with using the Onshape REST API. Please note that samples shown on this page are only designed to be used as a quick start guide.

All codes written on this page, with different programming languages, are all trying to make a GET request to the getPartStudioMassProperties API endpoint to retrive the mass property of the part in this public Onshape document. For more details about this API endpoint, please visit this endpoint on the Glassworks API explorer.

Languages demonstrated on this page:

  • Python
  • JavaScript
  • C/C++

Python

import requests 
import json 

# Use the API keys generated from the Onshape developer portal 
# Note: replace with your own keys 
api_keys = (ACCESS_KEY, SECRET_KEY)

# Assemble the URL for the API call 
api_url = "https://cad.onshape.com/api/partstudios/d/58d051743d98a90f86e9e2ab/w/17e0ba65df28e4e72f7f4961/e/4a289bd094a999329a3900af/massproperties"

# Optional query parameters can be assigned 
params = {}

# Define the header for the request 
headers = {'Accept': 'application/json;charset=UTF-8;qs=0.09',
           'Content-Type': 'application/json'}

# Putting everything together to make the API request 
response = requests.get(api_url, 
                        params=params, 
                        auth=api_keys, 
                        headers=headers)

# Print (formatted) response 
print(json.dumps(response.json(), indent=4))

JavaScript

import fetch from 'node-fetch'; 

// Define an async function for the API request 
async function getMass(url='') {
    const response = await fetch(url, {
      method: 'GET', // May change to GET, POST, or DELETE
      headers: {
        'Content-Type': 'application/json', 
        Accept: 'application/json;charset=UTF-8;qs=0.09', 
        // Replace with API keys generated from the Onshape developer portal
        Authorization: `Basic ${btoa('ACCESS_KEY:SECRET_KEY')}`
      }
      // May add body here for POST requests 
    });
    return response.json(); // Return the JSON response 
}

// Make the API request with the assembled URL 
// May add optional query parameters to the end of the URL in the form
// "?PARAM_NAME=PARAM"
getMass( 'https://cad.onshape.com/api/partstudios/d/58d051743d98a90f86e9e2ab/w/17e0ba65df28e4e72f7f4961/e/4a289bd094a999329a3900af/massproperties'
).then((data) => {
    console.log(data); // Display the JSON response 
});

C/C++

Before running the following script, you can first run the following command line in terminal to create a credential using your API keys generated from the Onshape developer portal.

$ echo -n ACCESS_KEY:SECRET_KEY | base64

#include <iostream>
#include <string>
#include <stdio.h>
using namespace std;

int main() {
    string url = "curl ";

    // Use "-X" to specify API type 
    url += "-X 'GET' "; 

    // Assemble the URL of the API 
    // May add optional query parameters to the end of the URL in the form
    // "?PARAM_NAME=PARAM" 
    url += "'https://cad.onshape.com/api/partstudios/d/58d051743d98a90f86e9e2ab/w/17e0ba65df28e4e72f7f4961/e/4a289bd094a999329a3900af/massproperties' "; 

    // Use "-H" to send headers 
    url += "-H 'accept: application/json;charset=UTF-8; qs=0.09' "; 
    // Replace CREDENTIAL with the Base64 credential generated from the command line above
    url += "-H 'Authorization: Basic CREDENTIAL'"; 
    
    // Use "-d" to send body (data) for POST requests 

    //Executing the cURL script
    system(url.c_str());
    return 0; 
}
