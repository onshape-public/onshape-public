Welcome to the Developer Portal

The Developer Portal is your place to create and administer applications, and find resources for developing an application that integrates with Onshape.

Getting Started

The API Overview document introduces the Onshape REST API, the lifeycle of an application, how your application will interact with Onshape and further resources for documentation and testing.

To further explore the API, use the Glassworks API Explorer tool.

Support

If you have questions about the Onshape REST API, have discovered a bug or need help send email to api-support@onshape.com

Sample Code

Sample code for using the Onshape REST API from a server application written in node.js is available at https://github.com/onshape-public
